Features

Interactive Game Builder

Create game scenes with an intuitive drag-and-drop interface. Add objects, set properties, and script behaviors without writing code.

Voxel Engine

Build and explore 3D voxel worlds with simple controls. Place blocks, destroy terrain, and navigate your creations in real-time.

Customizable Objects

Add and customize objects with various properties and behaviors. Create interactive elements that respond to player actions.

Export & Share

Export your creations and share them with others. Publish your games as standalone web applications or embed them in websites.

About HDMY5 Game Builder

HDMY5 Game Builder is a powerful and flexible game development framework that allows developers to create 2D and 3D games with ease. It provides a wide range of features and tools to streamline the game development process.

These demos showcase various capabilities of the HDMY5 Game Builder, from 3D voxel rendering to intuitive scene building. Our goal is to make game development accessible to everyone, regardless of their programming experience.
